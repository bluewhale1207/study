
* Command + delete 删除当前行
*  Command + b / Command + 点击函数 跳转到声明处
*  Command + [] 跳转到光标之前/之后的位置
*  Command+l 跳转到制定行
* Command + 展开
* Command +- 折叠
* Command + / 注释/取消注释
* Tab/shift+tab 缩进/取消缩进
* Command+f 当前文件搜索,enter 下一个/shift enter 上一个
* option+command+L（+↑） 所选代码格式化（当前文件）
* Command+z(+shift) 回退/redo
